# Retail Store Sales Data Mining Project

## Project Overview

This project focuses on analyzing a retail store sales dataset using Data Mining techniques.  
The main goal is to extract useful insights from sales transactions, customer behavior, payment methods, and product categories.

The project includes:
- Data Collection
- Data Preprocessing
- Data Mining Algorithms
- Data representation-Visualization
- Project Documentation

## Dataset Description

The dataset contains retail store sales transactions with information about:
- Product categories
- Item names
- Quantity purchased
- Price per unit
- Total amount spent
- Payment methods
- Store location
- Transaction dates
- Discount information

### Dataset Features

| Column Name | Description |
|---|---|
| Transaction ID | Unique transaction identifier |
| Customer ID | Unique customer identifier |
| Category | Product category |
| Item | Product name |
| Price Per Unit | Price of one item |
| Quantity | Number of purchased items |
| Total Spent | Total transaction amount |
| Payment Method | Payment type used |
| Location | Purchase location |
| Transaction Date | Date of transaction |
| Discount Applied | Whether a discount was applied |
